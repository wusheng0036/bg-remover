## 快速命令速查

```bash
# 开发模式
./quickstart.sh start      # 前台启动
./quickstart.sh daemon     # 后台启动
./quickstart.sh stop       # 停止
./quickstart.sh test       # 测试 API

# 生产模式
sudo ./deploy.sh install   # 安装
sudo ./deploy.sh start     # 启动
sudo ./deploy.sh status    # 状态
sudo ./deploy.sh logs      # 日志
sudo ./deploy.sh restart   # 重启

# 其他
./scripts/backup.sh        # 手动备份
./scripts/benchmark.sh     # 压力测试
pytest tests/              # 运行测试
```
