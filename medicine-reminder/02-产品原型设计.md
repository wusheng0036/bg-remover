## 🎨 设计规范

### 色彩
| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #07C160 | 微信绿，信任感 |
| 辅助色 | #FF9500 | 警告/待服药 |
| 成功色 | #07C160 | 已完成 |
| 危险色 | #FF3B30 | 漏服/警告 |
| 背景色 | #F5F5F5 | 页面背景 |

### 字体
- 标题：18px 加粗
- 正文：16px 常规
- 辅助文字：14px 灰色
- 老年模式：整体放大 1.5 倍

### 间距
- 页面边距：16px
- 卡片间距：12px
- 元素间距：8px

## 🔄 核心流程

### 添加药品流程
```
开始 → 点击添加 → 输入药品信息 → 设置时间 → 保存 → 订阅消息授权 → 完成
```

### 服药打卡流程
```
收到提醒 → 点击通知 → 打开小程序 → 点击服药 → 确认 → 打卡成功 → 通知家属 (可选)
```

### 邀请家人流程
```
点击邀请 → 生成邀请卡 → 微信分享 → 家人接受 → 绑定关系 → 开始接收通知
```

### 漏服处理流程
```
到点未打卡 → 15 分钟后二次提醒 → 30 分钟后通知家属 → 家属可电话提醒 → 用户补打卡
```
