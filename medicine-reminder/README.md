# 安心吃药 - 微信小程序

> 按时吃药，家人安心

## 📱 项目简介

一款专注于服药管理的微信小程序，核心功能：
- ✅ 智能服药提醒
- ✅ 一键打卡确认
- ✅ 家庭协同监督
- ✅ 依从性统计

## 🚀 快速开始

### 1. 环境准备

- 微信开发者工具（最新版）
- 开通微信云开发

### 2. 配置云开发

修改 `miniprogram/app.js` 中的云环境 ID：

```javascript
wx.cloud.init({
  env: 'your-env-id',  // 替换为你的云环境 ID
  traceUser: true,
});
```

### 3. 导入项目

1. 打开微信开发者工具
2. 导入项目，选择 `miniprogram` 目录
3. 填入你的 AppID

### 4. 上传云函数

在开发者工具中：
1. 右键 `cloudfunctions` 目录
2. 选择"上传并部署：云端安装依赖"
3. 等待部署完成

### 5. 创建数据库

在云开发控制台创建以下集合：
- `users` - 用户表
- `medicines` - 药品表
- `records` - 服药记录表
- `families` - 家庭成员表
- `reminder_tasks` - 提醒任务表

权限设置：所有集合设置为"仅创建者可读写"

### 6. 配置订阅消息

1. 登录微信公众平台
2. 申请订阅消息模板（参考 技术方案.md）
3. 将 templateId 填入云函数代码

### 7. 配置定时任务

为 `sendReminder` 云函数添加定时触发器：
- 触发器名称：`reminderTimer`
- 触发周期：`*/5 * * * *`（每 5 分钟）
