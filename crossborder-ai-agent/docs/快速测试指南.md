# 🚀 快速测试指南

> 5 分钟上手测试 4 个 Agent

---

## 📋 测试前准备

### 1. 确认环境

```bash
cd /root/.openclaw/workspace-ops/crossborder-ai-agent
node -v  # 应该显示 v24.14.0
```

### 2. 安装依赖（如果还没装）

```bash
npm install
```

### 3. 配置文件

```bash
# 确认配置文件存在
ls config/.env
```

---

## 🎯 开始测试

### 方式 1：命令行测试（推荐新手）

```bash
# 启动测试模式
node src/index.js
```

然后输入测试指令：

```
📥 输入消息：买家问 where is my order 怎么回复？
📤 回复：Hi! Your order will be shipped within 1-2 business days...

📥 输入消息：优化这个标题：Wireless Bluetooth Headphones
📤 回复：🎯 标题优化建议（3 个版本）...

📥 输入消息：监控这个竞品：amazon.com/dp/B08XXX
📤 回复：✅ 已添加监控任务...

📥 输入消息：生成销售周报
📤 回复：📊 销售周报...

📥 输入消息：quit
👋 再见！
```

---

### 方式 2：脚本测试（批量测试）

```bash
# 运行完整测试脚本
node -e "
const Dispatcher = require('./src/dispatcher.js');
const d = new Dispatcher();

// 测试客服
d.dispatch({ content: '买家问物流怎么回', sender: 'test' });

// 测试运营
d.dispatch({ content: '优化标题：Headphones', sender: 'test' });

// 测试监控
d.dispatch({ content: '监控竞品：amazon.com/dp/XXX', sender: 'test' });

// 测试数据
d.dispatch({ content: '生成销售报表', sender: 'test' });
"
```
