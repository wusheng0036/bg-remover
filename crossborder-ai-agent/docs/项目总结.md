## 🚀 快速启动

### 1. 配置环境

```bash
cd /root/.openclaw/workspace-ops/crossborder-ai-agent
cp config/default.env config/.env
# 编辑 config/.env 填入你的配置
```

### 2. 测试运行

```bash
# 测试调度器
node -e "
```

