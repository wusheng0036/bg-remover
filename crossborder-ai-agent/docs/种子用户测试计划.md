## 📝 反馈收集

### 每日反馈（群内）

```
【Day X 反馈收集】

各位种子用户好！

今天用得怎么样？有任何问题或建议都可以在群里说~

快速反馈：
👍 今天用了哪个功能？
💡 有什么觉得好用的地方？
🐛 遇到什么问题或 bug？
💰 这个功能你愿意付多少钱？

感谢大家的支持！
```

### 访谈问题（Day 6）

**使用体验**
1. 这 7 天里，你最喜欢哪个功能？为什么？
2. 哪个功能最没用？为什么？
3. 有没有哪个功能让你觉得"哇，这个好"？
4. 使用过程中遇到过什么困难吗？

**价值评估**
5. 这个工具帮你节省了多少时间？
6. 如果不用这个工具，你需要花多少钱雇人做这些事？
7. 你觉得这个工具值多少钱一个月？
8. 测试结束后，你愿意继续付费使用吗？

**改进建议**
9. 如果让你提一个改进建议，会是什么？
10. 你还希望增加什么功能？
11. 你会把这个工具推荐给朋友吗？为什么？

**基本信息**
12. 你做的什么平台？（亚马逊/Shopify/其他）
13. 团队多少人？
14. 目前用什么工具管理店铺？
