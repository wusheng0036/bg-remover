# 📱 飞书群聊使用指南

> 4 个机器人 + 1 个调度器 = 完整的跨境电商 AI 运营团队

---

## 🎯 建群步骤

### 1️⃣ 创建飞书群

1. 打开飞书 → 新建群聊
2. 群名建议：`跨境电商 AI 助手 - [你的店铺名]`
3. 群类型：内部群 或 外部群（根据团队成员选择）

---

### 2️⃣ 邀请机器人

把 4 个机器人都拉进群：

| 机器人 | 角色 | 识别名 |
|--------|------|--------|
| 机器人 1 | 客服 Agent | @客服助手 |
| 机器人 2 | 运营 Agent | @运营助手 |
| 机器人 3 | 监控 Agent | @监控助手 |
| 机器人 4 | 数据 Agent | @数据助手 |

---

### 3️⃣ 配置调度器

在群里发送第一条消息：
```
/agent init
```

调度器会回复：
```
✅ 跨境电商 AI 助手已就绪！
📊 当前状态：4/4 个 Agent 在线
💡 说"帮助"查看功能列表
```

---

## 💬 群聊指令

### 🤖 自动调度（推荐）

直接在群里说需求，系统自动分配：

```
"买家问物流怎么回复？"        → 客服 Agent
"帮我优化这个商品标题"          → 运营 Agent
"监控这个竞品的价格"            → 监控 Agent
"生成上周的销售报表"            → 数据 Agent
```

---

### 🎯 指定机器人

也可以 @ 具体机器人：

```
@客服助手 设置自动回复模板
@运营助手 写 10 个商品描述
@监控助手 添加竞品链接
@数据助手 分析这个品类趋势
```

---

### 📋 系统指令

| 指令 | 功能 |
|------|------|
| `/help` 或 `帮助` | 查看功能列表 |
| `/status` 或 `状态` | 查看系统运行状态 |
| `/config` | 查看当前配置 |
| `/reset` | 重置会话（清除上下文） |

## ⚙️ 高级配置

### 多店铺管理

如果有多个店铺，可以这样区分：

```
[店铺 A] 买家问物流怎么回复？
[店铺 B] 优化这个标题...
```

调度器会自动识别店铺前缀，调用对应配置。

---

### 定时任务

```
每天早上 9 点生成昨天的销售报表
每周一上午 10 点发送竞品周报
每月 1 号生成月度经营分析
```

在群里设置：
```
/reminder add 每天 9:00 生成销售报表
```

---

## 🚨 注意事项

1. **敏感操作确认** - 涉及价格调整、批量操作时会要求二次确认
2. **人工审核** - 重要回复（如退款承诺）会建议人工审核后再发送
3. **频率限制** - 避免短时间大量请求，防止 API 限流
4. **数据备份** - 所有配置和对话记录自动同步到飞书多维表格
